什么是HTML？

  • HTML 全称 Hypertext Markup Language 「超文本标记语言」
  • 不是编程编程语言 不做逻辑处理

  • 告诉浏览器如何构造网页

    HTML Elenment示例如下：

    <p> XXXXXX</p>

  • < p > 是起始标签 【Opening Tag】

  • < /p> 是结束标签 【Closing Tag】

  • “XXX” 是文本内容 【Conent】

HTML的基本构成示例：

<!DOCTYPE html>  （解释文档的类型）

<html>
      
      <head>
         <title> This is a page title </title>  (这是标题栏上的网页标题)    
      </head>    
      
      <body>
         <h1> This is a heading </h1>  （标题文字）
         <p> This is a garagraph </p>  （文本段落）
         <p> This is a garagraph </p>  （文本段落）
      </body>

</html>

块级元素和内联元素

块级元素

例如 < div> < h1>～< h6> < p> - 在页面以块的形式展现 - 出现在新的一行 - 占全部宽度

内联元素

例如 < a> < img> < em> < stong> - 通常在块级元素内 - 不会导致文本换行 - 只占必要的部分宽度
